# Fundamentals

## Components

> React components are JavaScript functions that return markup
>
> [React](https://react.dev/learn)

```jsx
function MyComponent() {
  return (
    <h1>...</h1>
  );
}

export default function MyApp() {
  return (
    <>
      <MyComponent />
      <p>...</p>
    </>
  )
}
```
